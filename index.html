Dropbox

Functionalities

  • Users will be able to upload files
  • Users will be able to download files
  • Users will be able to access their files by going to /uploaded/:name

User Interface

  • Button
  • Input

User Flow

  • Going to home page (method: GET)
    • get route
  • Uploading a file (method: POST)
    • post route
  • Getting the file (method: GET)
    • get route

Libraries

  • Built in: fs
    • Purpose: allows us to read and write files
    • fs.readFile(name, callback)
    • fs.writeFile(name, callback)
  • Built in: path
    • Purpose: ensure that we can grab the upload folder
  • Express
    • Purpose: handle routes
  • Body parser
    • Purpose: parse body into json format
  • Express-fileupload
    • Purpose: handle upload functionality
